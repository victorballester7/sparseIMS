### With Virtual Environments

1. Create environment

```bash
cd sparseIMS
python -m venv .venv
```

2. Activate environment

Linux/Mac - Bash

```bash
source .venv/bin/activate
```

Windows

```bash
.venv\Scripts\activate
```

3. Install Dependencies

```bash
pip install -e .
```

4. Test

```bash
python examples/dimension.py
```
